# Abbey Root

Abbey Root is my home Linux lab for learning, experimentation, automation, and AI-assisted development.

The lab is built around Proxmox, Linux virtual machines, Docker, Ansible, Git, and self-hosted services.

## Purpose

Abbey Root exists to build practical skills by creating real projects.

The long-term goal is to evolve the lab into a reproducible Infrastructure-as-Code platform managed through Git, Ansible, templates, and automated deployments.

## Current Focus

- Proxmox virtualization

- Ansible automation
- Generated documentation
