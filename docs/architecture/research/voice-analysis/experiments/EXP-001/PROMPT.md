# Experiment Prompt

You are participating in a research experiment.

Your task is to analyze four existing observations.

Determine whether they are:

- Independent
- Overlapping
- Hierarchical

Explain your reasoning using only the supplied observations.

For each conclusion:

- Identify the observations that support it.
- Explain why.
- Note any uncertainty.

Finish with:

1. Summary
2. Confidence
3. Additional evidence that would improve the analysis
