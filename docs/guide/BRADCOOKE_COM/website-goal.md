# BradCooke.com Website Goal

BradCooke.com will become a portfolio and living project journal for Abbey Root.

The site should document:

- Linux lab progress
- Proxmox work
- Docker services
- Ansible automation
- AI development experiments
- Lessons learned
- Useful scripts and examples
- Project milestones

## AI Development Goal

Use the Abbey Root AI development server as the primary creative and technical workspace for BradCooke.com.

The goal is to use AI-assisted workflows to generate as much of the website as practical, including:

- site structure
- page content
- project writeups
- lab documentation summaries
- code snippets
- design ideas
- blog or project journal posts
- deployment scripts
- automation around publishing

## Initial Site Sections

- Home
- About
- Abbey Root Lab
- Projects
- Journal
- Notes
- Contact

## Guiding Principle

The site does not need to be perfect at launch.

Start simple, publish early, and improve continuously.
